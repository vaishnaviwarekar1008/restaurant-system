# 🍽️ Restaurant Recommender System

## 📌 Description

This project is a Restaurant Recommendation System built using Machine Learning and Flask.
It recommends restaurants based on user preferences using a trained model.

---

## 🚀 Features

* Restaurant recommendation system
* Machine learning model integration
* Flask-based web application
* Simple and user-friendly interface

---

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Flask

---

## 📁 Project Structure

Restaurant-Recommender-main/
│
├── Flask/        # Flask application
├── Model/        # Model building script
├── Dataset/      # Dataset (not included)
├── README.md

---

## ⚙️ Setup Instructions

1. Clone the repository
   git clone https://github.com/aabidmulla/restaurant-recommender.git

2. Install dependencies
   pip install pandas numpy scikit-learn flask

3. Build the model
   cd Model
   python build_model.py

4. Run the Flask app
   cd ../Flask
   python app1.py

---

## 🌐 Output

Open in browser:
http://127.0.0.1:5000
